- мы можем выполнять другой код, если есть ошибка

```python
try:
    num1 = int(input())
    num2 = int(input())
    print(num1 / num2)
except ValueError:
    print('Программа принимает только числа')
except ZeroDivisionError:
    print('На ноль делить нельзя')
else:
    print('Блок try сработал без ошибки')
finally:
    print('Выполняю всегда в конце')
```
