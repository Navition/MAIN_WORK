```python
import turtle
p = turtle.Pen()
turtle.bgcolor('yellow')
p.shape('circle')   # square, trianlge, arrow, turtle
p.color('blue')     # цвет линии
p.pensize(3)        # ширина линии
p.forward(100)      # прочертить линию на 100 пикселей
p.circle(45)        # начертить круг с радиусом 45 пикселей
p.left(90)          # повернуть влево на 90 градусов
p.right(45)         # вправо на 45 градусов
```
