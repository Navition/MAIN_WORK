- элементы не по порядку
- нельзя обратиться по индексу
- нет дубликатов

```python
st = set() # создание пустого множества
st = {'яблоко', 'банан', 'апельсин', 'груша'}
st.add('ананас')            # добавление
st.discard('апельсин')      # удаление
st2 = {"клубника", "виноград"}
st.update(st2)              # объединение множеств
```
