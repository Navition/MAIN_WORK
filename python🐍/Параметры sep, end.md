## sep
```python
a = 1
b = 2
c = 3
print(a, b, c)      # выводит несколько знчение через пробел
# вместо пробела можно использовать другой символ, добавив в конце sep='*' (можно указать другой символ)
print(a, b, c, sep='**')
Выведит
a**b**c
```
## end
```python
a, b, c = 4, 5, 6       # можем создать несколько переменных сразу
print(a, b)
print(c)
# сколько принтов, столько и строчек выводит, однако это тоже можно изменить, добавив end='@'
print(a, b, end='&&')
print(c)
# всё записалось в одну строчку
# по-умолчанию используется перенос строки (end='\n')
```

```python
print("a", "b", "c", end="**")
Выведит
a b c**
```
